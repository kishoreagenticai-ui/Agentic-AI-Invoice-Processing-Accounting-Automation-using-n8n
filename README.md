# Agentic-AI-Invoice-Processing-Accounting-Automation-using-n8n
Agentic AI Invoice Processing &amp; Accounting Automation using n8n, OCR, Gemini AI, Supabase and Gmail APIs.

An Agentic AI workflow that automatically processes invoices received via email.

Features:
- Gmail invoice detection
- OCR extraction
- AI-powered invoice parsing
- Validation checks
- Expense categorization
- Database storage (Supabase)
- Approval notifications
- Google Sheets reporting
